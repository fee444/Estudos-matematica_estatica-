# 📘 Laboratório de Vetores e Espaço Vetorial

Este laboratório aborda conceitos fundamentais relacionados a vetores e espaço vetorial.

## 📚 Conteúdo

- Adição de vetores  
- Subtração de vetores  
- Multiplicação de vetores  
- Produto vetorial (cross product)  
- Produto escalar (dot product)  
- Transposta de vetor  

## 💻 Tecnologias utilizadas

Além do conteúdo teórico, também é praticado o uso de Python com as seguintes bibliotecas:

- NumPy  
- Matplotlib  
- Pandas  

## 🎯 Objetivo

Reforçar os conceitos matemáticos de vetores, aplicando-os na prática com Python para análise e visualização de dados.
